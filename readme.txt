To run the project launch it in the cloud SDK environment by navigating to the directory this is located in and running
"dev_appserver.py app.yaml".

To like a post go to the url /like/(blog ID)
	In (blog ID) put the ID of the blogpost you wish to like.  Users cannot like their own posts/
To edit a post go to the url /blog/edit/(blog ID)

to delete a post go to the url /blog/delete/(blog ID)

to logout go to the url /logout and you will be logged out and redirected to the home page where you can create an account or sign in.
